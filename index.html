Features

Everything you need to review code smarter

Built on nikic/php-parser and PHPMetrics. No external services, no accounts — just run it.

Risk Scoring
Per-file risk scores from cyclomatic complexity, LLOC, and parameter counts. Configurable HIGH / MED / LOW / INFO thresholds per project.
AST-Powered
Built on PHP-Parser for true AST analysis — accurate method-level metrics, not approximations. Works on complex Laravel codebases.
GitHub PRs
Analyze any PR with --pr=<url>. Integrates into GitHub Actions for automated analysis on every pull request.
Architecture Viz
Interactive force graphs, layered cake views, and dependency maps. Visualize your codebase structure and where layer boundaries break.
Multiple Formats
HTML, Markdown, JSON, and detailed metrics exports. Pipe to CI logs, post to PR comments, or build tooling on the JSON output.
Custom Rules
Define your own layer stack and architectural patterns. Pin watched files to surface critical paths at the top of every report.

Install

Up and running in three commands

Requires Laravel 11–13 and PHP 8.2+.

Install 
$ composer require vistik/laravel-code-analytics
$ php artisan vendor:publish --tag="laravel-code-analytics-migrations"
$ php artisan migrate
Usage 
$ php artisan code:analyze
$ php artisan code:analyze --pr=github.com/org/repo/pull/42
$ php artisan code:analyze --format=html --open
$ php artisan code:analyze --all --format=json
